# Web Letter: A Digital Greeting

A simple, charming, and interactive digital letter experience built with HTML, CSS, and Vanilla JavaScript. Perfect for sending a thoughtful message to someone special.

## Features
- **Interactive Envelope:** The letter is tucked inside an animated envelope that opens upon clicking.
- **Smooth Animations:** Features CSS animations for opening, closing, and particle effects to enhance the experience.
- **Responsive Layout:** Designed to work across different screen sizes.
- **Easy Customization:** Simply edit the HTML to update the message content.

## Getting Started
1. Clone this repository.
2. Open `letter.html` in your web browser.
3. Click the envelope to reveal the letter content.

## Customization
To personalize the letter, open `letter.html` in a code editor and find the `letter-content` div. You can update the following:
- **Greeting:** Edit the `<p class="greeting">` tag.
- **Body Content:** Add or remove `<p>` tags with your own message.
- **Signature:** Update the `<p class="signature">` tag.
